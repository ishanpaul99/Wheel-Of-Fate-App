# Wheel of Fate

Wheel of Fate is a fantasy-inspired decision-making web application built using HTML, CSS, and JavaScript. Users can enter their own choices, spin a magical wheel, and allow fate to determine the final outcome.

Featuring animated wheel spins, mystical visuals, local storage history tracking, and a responsive design, the application transforms ordinary decisions into an entertaining and immersive experience.

## Features

- Add unlimited custom choices
- Interactive spinning wheel
- Smooth wheel spin animation
- Randomized outcome selection
- Dynamic wheel generation
- Fate history tracking
- Local Storage support
- Fantasy-themed user interface
- Magical visual effects and animations
- Responsive design for desktop and mobile devices
- Smooth scrolling navigation
- Modern and immersive user experience

## How to Use

1. Enter two or more choices.
2. Click **Add Choice** to include them in the wheel.
3. Press **Spin The Wheel**.
4. Watch the wheel spin and gradually slow down.
5. Discover the option chosen by fate.
6. View previous outcomes in the Fate History section.
